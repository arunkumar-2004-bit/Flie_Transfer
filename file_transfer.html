How It Works

Files go directly from your browser to GitHub via API

1️⃣
Configure once
Enter your GitHub token, username, and repo name. Saved locally in your browser.
2️⃣
Drop any files
Drag and drop or click to select. Supports all file types. Multiple files at once.
3️⃣
Upload to GitHub
Files are uploaded via GitHub API directly from your browser. No middleman server.
4️⃣
Get public link
Instantly get a raw.githubusercontent.com link. Share with anyone — no login needed.
📌 Public link format https://raw.githubusercontent.com/USERNAME/REPO/BRANCH/FOLDER/filename.ext
Anyone with this link can download the file directly — no GitHub account needed.
